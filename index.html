Indeed Web Crawler

This is a simple CRUD application that crawls indeed.com for front-end engineer job postings using Node.js and Angular.js. Each job title you see in the screen shot is the "title" and "URL" pulled from indeed.com which the user can click through to and apply. The user can also add notes to each job entry once added to the job list and update those notes as they go.

Technology

Indeed Web Crawler was built using node.js and angular.js. Bootstrap, CSS3 and HTML5 were also used.

CSV to MySQL Importer

This is a node CLI application I created to help automate the import of client product files (CSV) into a mySQL database. Repository includes instructions on how to run this application along with a sample file to run against.

Technology

This importer was built primarily in Node, which utilizes the NPM packages csv-parse, mysql and fs. MySQL was used to store the data after import is complete.
